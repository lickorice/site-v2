Stuff I did for university

Paging behavior manipulation in xv6

CS 140 - OPERATING SYSTEMS

xv6 is a teaching operating system developed by MIT, implemented in ANSI C. As students of CS 140, we are tasked to appropriately alter its source code to make its paging procedure much more efficient.

We had to alter kernel-level code to implement custom system calls, such as printing the page table, and other system variables. We also had to implement lazy page allocation and empty page reuse to optimize compute time and space.

Single cycle MIPS processor in SystemVerilog

CS 21 - COMPUTER ORGANIZATION AND ARCHITECTURE

SystemVerilog is a hardware description language (HDL) used in designing processors and other electronic systems, such as integrated circuits.

We had to come up with cost-efficient designs and solutions for implementing standard and customized 32-bit MIPS instructions for this project. We used Xilinx Vivado for the designing and simulation of the project.

Peg solitaire solver in 32-bit MIPS

CS 21 - COMPUTER ORGANIZATION AND ARCHITECTURE

MIPS is a RISC microprocessor architecture used in small computing appliances such as routers and smaller computers. It is also used in the original Sony Playstation, the PSP, and other early Nintendo products.

We are tasked to make a peg solitaire solver from scratch using the assembly language. My solution involved a recursive backtracking solution.

Simple game in Python 3 using pyglet

CS 11 - COMPUTER PROGRAMMING I

Python is a high-level, general-purpose, interpreted programming language. The project given to us involves making a game using basic concepts (conditionals, loops, objects, classes) of programming. We had creative freedom in doing it.

A Filipino's Guide to Programming

A Filipino's Guide to Programming is a YouTube video series I have worked on since February of 2021. This project of mine was conceived from the realization of the lack of readily-available, quality, and digestible intellectual Filipino on the video platform. The series is delivered in Filipino-English (Taglish).

As of now, it covers topics from introductory programming classes, such as basic data structures (arrays, pointers), algorithm analysis, and different problem-solving paradigms (backtracking, dynamic programming). In the near future, more videos about more advanced data structures (trees, graphs, etc.) and algorithms will be uploaded.

The channel has a little over 2,400 subscribers, with a total of 32,000 views and counting. All of my videos can be viewed for free.
